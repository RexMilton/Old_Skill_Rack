Count Duplicates [AZ-R1] (Id-2947)

Accept an Array of size N and print the total number of duplicate elements (The elements which occur two or more times).

Input Format:
The first line contains N.
The second line contains the N positive integer values, each separated by a space.

Output Format:
The first line contains the count of duplicate elements.

Boundary Conditions:
1 <= N <= 1000

Example Input/Output 1:
Input:
8
1 3 1 4 5 6 3 9

Output:
2

Explanation:
1 and 3 occur two or more times.

Max Execution Time Limit: 5000 millisecs
